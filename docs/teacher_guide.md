# Инструкция для педагогов — бот ТСК ФОКУС

Бот помогает педагогу вести учёт занятий, учеников и зарплаты через Telegram. Вся информация хранится централизованно и доступна администрации школы.

## Как начать

1. Открой бота в Telegram: `@EkaterinaReka_bot`.
2. Отправь команду `/start`.
3. Откроется главное меню педагога.

Если бот отвечает «Нет доступа» — значит, администратор ещё не добавил тебя как педагога. Напиши администратору свой Telegram ID (его можно узнать у бота `@userinfobot`).

## Главное меню педагога

- 💃 **Мои пары** — список твоих учеников, поставленных в пары, и управление партнёрами.
- 👩‍🎓 **Мои ученики (соло)** — список учеников, с которыми ты занимаешься индивидуально (без пары), и добавление новых учеников.
- 🏢 **Мои группы** — тренировочные группы, которые ты ведёшь, с составом учеников.
- 📋 **Мои занятия** — добавление, просмотр и удаление проведённых занятий.
- 📊 **Моя статистика** — сводка по часам и заработку за период.
- 📤 **Сдать период** — финализация месяца для расчёта зарплаты.

## 📋 Мои занятия

### Добавить занятие

1. **Мои занятия** → **➕ Добавить занятие**.
2. Выбери дату (календарь или «Сегодня»).
3. Выбери тип занятия:
   - 👥 **Групповое** — занятие с тренировочной группой.
   - 💃 **Парное** — индивидуалка с одной или несколькими парами.
   - 👤 **Индивидуальное (соло)** — с одним или несколькими соло-учениками.
4. Выбери длительность: 35 / 45 / 60 / 90 мин.
5. Дальнейшие шаги зависят от типа (ниже).
6. В конце — подтверди сохранение (💾).

#### Групповое занятие

- Если у тебя **одна группа** — сразу попадёшь на шаг отметки присутствия.
- Если **несколько групп в одном филиале** — выбери группу.
- Если **группы в разных филиалах** — сначала выбери филиал, потом группу.
- На экране «Отметить присутствующих?»:
  - **✅ Отметить** — откроется чек-лист учеников группы, поставь ✅ напротив присутствовавших. Есть кнопка **☑️ Отметить всех** / **◻️ Снять всех**.
  - **⏭ Пропустить** — занятие сохранится без привязки учеников (но группа и факт занятия записываются).
- **💾 Подтвердить** — занятие сохранено.

#### Парное занятие

- Покажется список твоих пар (чекбоксы).
- Отметь пары, которые были на занятии → **💾 Подтвердить**.

#### Индивидуальное (соло)

- Список твоих соло-учеников (чекбоксы).
- Отметь тех, с кем было занятие → **💾 Подтвердить**. Для каждого будет создано отдельное занятие.

### Посмотреть занятия

**Мои занятия** → **👁 Посмотреть занятия** → выбери фильтр:
- **Сегодня** — сегодняшние занятия.
- **За месяц** — список месяцев с занятиями.
- **Все** — все занятия.
- **Выбрать дату** — через календарь.

Кликни по занятию, чтобы открыть карточку (дата, тип, длительность, группа, ученики).

### Удалить занятие

**Мои занятия** → **🗑 Удалить занятие** → фильтр → список → тап по занятию → **🗑 Удалить занятие** в карточке.

Редактировать занятие нельзя: если ошибся — удали и создай заново.

Занятия из **сданного периода** защищены: рядом 🔒, кнопки удаления нет. Если нужно исправить — обратись к админу.

## 💃 Мои пары

Показывает всех твоих учеников, которые состоят в парах друг с другом.

- Кликни по паре → **🔄 Изменить партнёра** или **❌ Убрать партнёра**.
- **➕ Создать пару** — выбери двух учеников из своего списка.
- Если у твоего ученика партнёр у другого педагога — управление парой только у админа.

## 👩‍🎓 Мои ученики (соло)

Список учеников, с которыми ты занимаешься индивидуально.

- **Карточка ученика** — имя, партнёр, ID.
- **🚪 Убрать из моего списка** — снять связь (ученик остаётся в базе).
- **➕ Добавить ученика** — два пути:
  - **Найти существующего** — поиск по имени → выбор из базы → привязка к тебе.
  - **Создать нового** — ввод ФИ → выбор группы → отправка заявки администратору. После одобрения ученик создастся и привяжется к тебе автоматически.

### Правила для ученика

- У ученика ровно одна группа (или нет группы).
- Группу задаёт админ или педагог при создании.
- Изменить группу можно через «Мои группы → Изменить состав» (см. ниже) или через админа.

## 🏢 Мои группы

Показывает твои тренировочные группы с филиалами.

- Тап по группе → карточка:
  - Филиал / название группы.
  - Список учеников.
- **✏️ Изменить состав** — чек-лист:
  - ✅ — ученик в группе (тап снимает из группы).
  - ☐ — ученик без группы (тап добавляет в твою группу).
  - Ученики из чужих групп не показываются — их перенос делает админ вручную.
- **💾 Готово** — вернуться в карточку группы.

Группу нельзя создать самому — это делает админ.

## 📊 Моя статистика

Выбери месяц → покажет:
- Количество занятий.
- Часы по типам.
- Заработок (ставка × длительность, пропорционально 45 мин).

Пока период не сдан — это **предварительная** сумма.

## 📤 Сдать период

Когда все занятия за месяц записаны:

1. **Сдать период** → выбери месяц.
2. Покажется сводка: кол-во занятий, суммарный заработок.
3. **💾 Подтвердить сдачу** — период блокируется, занятия нельзя удалять. Админ увидит заявку и проведёт оплату.

Сдавать можно **только после окончания месяца** (или если так решено администратором).

## Частые вопросы

**Я ошибся в дате/длительности занятия.**
Удали занятие и создай заново. Правка полей не поддерживается намеренно.

**Ученика нет в ростере группы.**
Значит, у него не проставлена эта группа. Открой «Мои группы → Изменить состав» и добавь (если он без группы) либо попроси админа перенести из другой группы.

**Ученик есть в моих соло, но я его не вижу на групповом занятии.**
Правильно: групповое занятие показывает только учеников выбранной группы. Сольная связь («Мои ученики») и группа — независимые вещи.

**Я не могу открыть карточку ученика, которого отметил на группе.**
Карточка доступна только если ученик в твоём списке «Мои ученики» или «Мои пары». Это защита — чтобы педагог не трогал чужих учеников без прямой связи.

**Сданный период — можно откатить?**
Только через админа (вручную в базе).

**Как добавить новую тренировочную группу, филиал, ставку?**
Эти операции делает администратор.

## Что делать, если что-то не работает

1. Проверь, что бот онлайн — отправь `/start`.
2. Если кнопка не отвечает — попробуй закрыть диалог и открыть заново.
3. Если видишь ошибку «Период сдан», «Группа не найдена» и т.п. — прочти сообщение, обычно оно объясняет причину.
4. Если ничего не помогает — напиши администратору с описанием что делал и что увидел.
