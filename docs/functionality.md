# Функционал фокус-бота — текущее состояние

## Роли
- **Админ** (`is_admin=True`) — управляет всеми сущностями.
- **Педагог** (`teacher_id is not None`) — отмечает свои занятия, управляет своими учениками.
- **Админ+педагог** — видит обе роли, переключается через «🔄 Режим …».
- **Новый пользователь** — `/start` авторегистрирует, админы получают push с кнопкой «Добавить как педагога».

## Общие команды
- `/start` — вход, регистрация, выбор режима.
- `/menu` — возврат в главное меню из любой FSM-точки (стейт очищается).

## 🔧 Админ

**Педагоги**
- Список, карточка (ставки, группы).
- Добавить (tg_id + имя) / удалить.
- Ставки: групповое / инд. педагогу / инд. ученику.
- Привязка к тренировочным группам.

**Ученики**
- Поиск по имени (с пагинацией, экранированием `_`/`:` в запросе).
- Списки: пары / солисты (сгруппированы по тренировочным группам).
- Добавить (с выбором группы) / удалить (пагинация 20/стр).
- Карточка: назначить партнёра / убрать партнёра.

**Заявки педагогов** (`📝 Заявки`)
- Список pending-заявок на новых учеников.
- Одобрить → создать нового / привязать к существующему; отклонить.
- Защита от race: PENDING-check + атомарный `mark_resolved`.
- **Хранятся в Google Sheets** `student_requests` — переживают рестарт.

**Филиалы и группы**
- CRUD филиалов.
- CRUD тренировочных групп внутри филиала.
- В группе: ростер учеников (чекбоксы), назначение педагогов.

**Счета и оплаты**
- `🧾 Счёт за период` — выбор период → группа → ученик → отправить счёт ученику.
- `💾 Подтвердить оплату` — выбор неоплаченных счетов → подтверждение.
- Навигация «Назад» сохраняет контекст группы/периода.

**Зарплаты**
- Выбор педагог → период → детализация начислений.
- Навигация «Назад» сохраняет контекст педагога.

**Редактирование занятия**
- Поиск занятия по ID → удаление (даже из сданного периода).

**Диагностика** — техническая вкладка (состояние кэша, пр.).

## ✏️ Педагог

**Отметить занятие** (FSM с «Отменой» и «Назад» на каждом шаге)
- Выбор даты (календарь).
- Тип: `👥 групповое` / `💃 парное` / `👤 сольное`.
- Групповое: выбор филиала → группы → (опционально) чекбокс-отметка присутствующих.
- Парное: мульти-выбор пар чекбоксами.
- Сольное: мульти-выбор учеников.
- Длительность: 35/45/60/90 мин.
- После сохранения: «продолжить с тем же днём» или «В меню».

**Мои пары / Мои солисты / Мои группы**
- Списки своих учеников, сгруппированные.
- Карточка пары: изменить/убрать партнёра, убрать из списка.
- Карточка солиста: переименовать, убрать из списка.
- Запрос на нового ученика → уходит админам.

**Мои занятия**
- Список с фильтрами: по дате, по месяцу, все.
- Карточка: удалить занятие (если период не сдан — иконка 🔒).

**Моя статистика**
- Сводка занятий/часов за период.

**Сдать период**
- Выбор месяца → подтверждение → занятия блокируются от редактирования.

## 🗄️ Данные (Google Sheets)
`users`, `teachers`, `students`, `branches`, `groups`, `teacher_groups`, `teacher_students`, `lessons`, `payments`, `submitted_periods`, `student_requests`.

Кэш: TTL 300с, инвалидация на каждую запись.

## 🚀 Инфраструктура
- **Хостинг**: Timeweb VPS (`147.45.146.247`), systemd-юнит `fokus-bot`.
- **Деплой**: `./scripts/deploy.sh` (rsync + restart).
- **FSM storage**: MemoryStorage (состояния не переживают рестарт, это ок).
- **Логи**: handler-timing и sheets-op timing в journald (`journalctl -u fokus-bot`).
