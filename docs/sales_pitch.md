# Бот «ФОКУС» — CRM для танцевальной студии прямо в Telegram

Один инструмент для учёта занятий, зарплат педагогов и счетов учеников. Без приложений, без новых платформ — всё через Telegram, которым педагоги и родители пользуются каждый день.

---

## Кому это нужно

Руководителям танцевальных студий и школ, которые устали от:

- таблиц Excel, которые каждый педагог ведёт по-своему;
- споров о количестве проведённых занятий в конце месяца;
- ручного пересчёта зарплат по разным ставкам (групповые / индивидуальные / парные);
- путаницы с тем, кто из учеников в какой паре, в какой группе, в каком филиале;
- забытых счетов родителям и «а я же заплатил в прошлом месяце».

Бот убирает эту рутину. Педагог отмечает занятие в Telegram за 30 секунд — администрация видит всё в реальном времени.

---

## Что умеет бот уже сегодня

### Для руководителя / администратора

**👨‍🏫 Педагоги**
- Добавление и удаление педагогов по Telegram ID.
- Индивидуальные ставки для каждого: групповое / индивидуальное педагогу / индивидуальное ученику.
- Карточка педагога с составом групп, которые он ведёт.
- Назначение педагога на группы (несколько групп, несколько филиалов).

**👩‍🎓 Ученики**
- Общая база учеников школы, единая для всех педагогов.
- Поиск по имени, карточка ученика.
- Управление парами: назначить партнёра, изменить, убрать.
- Список всех пар и всех солистов в один клик.
- Модерация заявок от педагогов на новых учеников (педагог не создаёт учеников в обход администрации).

**🏢 Филиалы и группы**
- Несколько филиалов — в каждом свои тренировочные группы.
- Состав группы — привязанные к ней ученики.
- Переименование, добавление, удаление — всё в Telegram.

**💰 Зарплаты**
- Автоматический расчёт зарплаты педагога за период.
- Формула: ставка × длительность, пропорционально стандартным 45 минутам.
- Раздельный учёт типов занятий (групповое / парное / сольное).
- Педагог сам «сдаёт период» — администратор видит готовую заявку.

**🧾 Счета учеников**
- Автоматический расчёт счёта ученика за период по факту посещения.
- Подтверждение оплаты — фиксируется в базе.
- Прозрачно для родителей (в клиентской части, см. ниже).

**✏️ Редактирование занятий**
- Правка занятий педагога, в том числе уже сданных (разблокировка по решению администратора).

**🔧 Диагностика**
- Проверка целостности данных, технические инструменты.

### Для педагога

Отдельная инструкция: [docs/teacher_guide.md](fokus-bot/docs/teacher_guide.md).

- 📋 **Занятия** — добавить за 3–4 тапа (дата → тип → длительность → участники).
- 💃 **Пары** — управление своими ученическими парами.
- 👩‍🎓 **Соло-ученики** — индивидуальные занятия, добавление новых учеников (через заявку админу).
- 🏢 **Группы** — список своих групп, редактирование состава.
- 📊 **Статистика** — часы и предварительный заработок за любой месяц.
- 📤 **Сдать период** — финализация месяца в один клик.
- 🔒 **Защита от ошибок** — сданные периоды нельзя править задним числом.

## Что это даёт бизнесу

| Было | Стало |
|---|---|
| Таблица занятий у каждого педагога своя | Единая база, один источник правды |
| В конце месяца 2 дня собираем цифры | Зарплата считается автоматически по клику |
| Родители звонят «сколько мы должны?» | Счёт формируется за секунду |
| Педагог забыл записать занятие — спорим | Запись в моменте, в Telegram, где он всё равно сидит |
| Новый филиал — новая Excel-простыня | Филиал добавляется в боте, данные не дублируются |
| Смена педагога — неделя передачи дел | Вся история в системе, группы переназначаются в пару кликов |

## Что скоро появится — клиентская часть

Сейчас бот закрывает потребности **педагогов и администрации**. Следующий большой блок — **ученики и родители**:

- Личный кабинет ученика/родителя в том же Telegram-боте.
- Расписание занятий и посещаемость — прозрачно, в реальном времени.
- История оплат, текущий баланс, напоминания об оплате.
- Автоматические уведомления о переносах и отменах занятий.
- Запись на пробные занятия и мероприятия.

Это превратит бота из внутреннего инструмента студии в единую точку взаимодействия со всеми участниками: педагог ↔ администратор ↔ ученик/родитель.

---

## Почему Telegram, а не отдельное приложение

- **Ноль порога входа** — педагоги и родители уже в Telegram, ничего ставить не надо.
- **Мгновенная готовность** — новый педагог начинает работать в день найма: получил доступ → открыл бота → поехали.
- **Работает на всём** — телефон, планшет, компьютер, часы. Кроссплатформенно по умолчанию.
- **Не ломается в неподходящий момент** — Telegram работает, значит, работает CRM.
- **Бесплатно для пользователей** — школа не платит за лицензии на каждого педагога.

---

## Технологическая надёжность

- **Google Sheets как база данных** — администрация всегда может открыть таблицу и всё увидеть/выгрузить.
- **Никаких «чёрных ящиков»** — данные прозрачны, никакой vendor lock-in.
- **Кэширование и оптимизация** — бот быстрый даже на тысячах записей.
- **Защита ролей** — педагог видит только своих учеников, администратор видит всё.
- **Защита от ошибок** — сданные периоды блокируются, правки только через админа.

---

## Как начать

1. Развёртывание — под ключ, под вашу школу.
2. Настройка филиалов, групп, ставок педагогов.
3. Инструктаж педагогов (15 минут — бот интуитивный).
4. Первый месяц — поддержка и корректировки по обратной связи.

Готов показать в работе на реальных данных.
